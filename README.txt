WWW-Kickstarter

This distribution provides access to Kickstarter's private API
to obtain information about your account, other users and and projects.


INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install


DEPENDENCIES

This module requires these other modules and libraries:

    Module::Build    (For installation only)
    FindBin          (For testing only)
    Test::More       (For testing only)
    autovivification
    Carp
    overload
    Time::HiRes
    URI
    URI::Escape
    URI::QueryParam
    version

    The default HTTP client requires the following:
        HTTP::Headers
        HTTP::Request::Common
        LWP::Protocol::https
        LWP::UserAgent

    The default JSON parser requires the following:
        JSON::XS
